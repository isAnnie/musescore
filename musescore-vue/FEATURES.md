# MuseScore Vue 乐谱编辑器增强功能

## 新增功能概述

本项目在原有基础上增强了乐谱编辑功能，提供了更加完整的音乐创作体验。

## 主要新增组件

### 1. NoteEditor.vue - 音符编辑器
- **功能**：提供完整的音符属性编辑界面
- **特性**：
  - 音符类型选择（全音符、二分音符、四分音符等）
  - 音高选择（支持所有八度音阶）
  - 时值调节（支持0.25拍到4拍）
  - 升降号设置
  - 装饰音添加（断奏、重音、保持、延长）

### 2. DraggableNote.vue - 可拖拽音符
- **功能**：实现音符的可视化拖拽编辑
- **特性**：
  - 支持鼠标和触摸拖拽
  - 实时位置更新
  - 音符选中状态管理
  - 边界检测和约束

### 3. HistoryManager.vue - 历史记录管理
- **功能**：提供撤销/重做功能
- **特性**：
  - 自动保存编辑历史
  - 支持Ctrl+Z/Ctrl+Y快捷键
  - 最多保存50步历史记录
  - 状态恢复功能

### 4. AudioPlayer.vue - 音频播放器
- **功能**：集成音频播放功能
- **特性**：
  - 基于Tone.js的音频引擎
  - 播放控制（播放、暂停、停止）
  - 进度条和时间显示
  - 音量调节
  - 播放模式选择（顺序、单曲循环、全部循环）
  - 播放速度调节

## 功能改进

### ScoreCanvas.vue 增强
- 集成了上述所有新组件
- 改进了布局和用户体验
- 添加了实时状态反馈
- 优化了响应式设计

### 数据模型扩展
- 扩展了Note接口，支持更多属性
- 添加了装饰音、力度记号等专业音乐符号支持

## 使用说明

### 基本操作流程
1. **创建乐谱**：点击"新建乐谱"按钮
2. **添加音符**：在画布上点击添加音符，或使用左侧工具栏
3. **编辑音符**：选中音符后，在右侧属性面板进行详细编辑
4. **移动音符**：直接拖拽音符到新位置
5. **播放预览**：使用右侧音频播放器试听效果
6. **撤销操作**：使用Ctrl+Z或工具栏按钮撤销更改

### 快捷键
- `Ctrl + Z`：撤销
- `Ctrl + Y`：重做
- `空格键`：播放/暂停（计划中）

## 技术栈

- **Vue 3** + TypeScript
- **Tone.js**：音频处理
- **Tailwind CSS**：样式框架
- **Vite**：构建工具
